# 🧠 PsyBot — Телеграм-бот для психолога

## Что умеет бот

- 🌐 Русский и украинский языки
- 📅 Запись на консультацию (выбор из слотов)
- 🔔 Уведомление психолога о новой записи
- 🌬 5 практик: дыхание 4-7-8, коробочное, заземление, когнитивное переосмысление, дневник эмоций
- 🆘 SOS-режим: 3 шага успокоения → запись на консультацию
- 👩‍⚕️ Команды для психолога: `/bookings` и `/slots`

---

## Шаг 1 — Создать бота в Telegram

1. Открой [@BotFather](https://t.me/BotFather) в Telegram
2. Напиши `/newbot`
3. Придумай имя и username (например `Anna_Psych_Bot`)
4. Скопируй **токен** — он выглядит как `7412345678:AAFxxx...`

---

## Шаг 2 — Узнать свой Telegram ID (для жены)

1. Открой [@userinfobot](https://t.me/userinfobot)
2. Напиши `/start`
3. Скопируй число из поля `Id:` — это и есть ADMIN_ID

---

## Шаг 3 — Настроить бота

Открой файл `config.py` и замени:

```python
ADMIN_ID = int(os.getenv("ADMIN_ID", "123456789"))  # ← ID жены
```

Также можно изменить слоты в `SLOTS`.

---

## Шаг 4 — Деплой на Railway (бесплатно)

1. Создай аккаунт на [railway.app](https://railway.app)
2. Нажми **New Project → Deploy from GitHub repo**
3. Загрузи папку бота (или подключи GitHub репозиторий)
4. Перейди в **Variables** и добавь:
   - `BOT_TOKEN` = твой токен от BotFather
   - `ADMIN_ID` = Telegram ID жены
5. Нажми **Deploy** — бот запустится автоматически!

---

## Команды психолога (жены)

| Команда | Что делает |
|---|---|
| `/bookings` | Показывает все записи |
| `/slots` | Показывает свободные и занятые слоты |

## Структура файлов

```
psybot/
├── bot.py          # Запуск бота
├── config.py       # Настройки (токен, ID, слоты)
├── storage.py      # Хранилище записей
├── texts.py        # Все тексты на RU и UK
├── requirements.txt
├── Procfile        # Для Railway
└── handlers/
    ├── client.py   # Главное меню, запись
    ├── practices.py # Практики
    ├── sos.py      # SOS-режим
    └── admin.py    # Команды психолога
```
