# Папка для картинок бота

Положи сюда свои фото и назови их точно так:

| Файл | Где показывается |
|---|---|
| `welcome.jpg` | Приветствие / главное меню |
| `breathing.jpg` | Дыхательные практики |
| `sos.jpg` | SOS экран |

## Требования к фото
- Формат: JPG или PNG
- Размер: до 5 МБ
- Соотношение сторон: лучше квадрат или горизонталь

## Где взять красивые бесплатные фото
- https://unsplash.com — ищи: "calm nature", "meditation", "mindfulness"
- https://www.pexels.com — то же самое, всё бесплатно
